# WifiProject
“내 위치 기반 공공 와이파이 정보를 제공하는 웹서비스 개발”

: 공공 데이터 자원을 끌어와서 내가 서비스 하고자 하는 데이터로
마이그레이션하여 웹을 기반으로 하여 원하는 형태로 출력
: 또한, 사용한 히스토리에 대해서 데이터베이스에 저장하는 기능과 이를 출력하는 기능을 구현
